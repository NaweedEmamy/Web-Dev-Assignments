Z-index

Z Index ( z-index ) is a CSS property that defines the order of overlapping HTML elements. Elements with a higher index will be placed on top of elements with a lower index.

Example

Box 1
Box 2
Box 3
Box 4

In the example above, it can be clearly seen that Box 2 has moved to the laft and would normally overlap Box 1, however, with a negative z-index, Box 2 has not done so. Instead, Box 1 has overlapped Box 2 (Default z-index of Box 1 =0)
