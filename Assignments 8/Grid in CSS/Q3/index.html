Grid Auto Rows/Columns

grid-auto-rows and grid-auto-columns are used when we do not explicitly specify the heigt/width of rows/columns. This way, the created row/column will be have an autoamic size.

Consider above grid cells. Here is the code snappit for the above grid's rows:

grid-template-rows: 100px;
grid-auto-rows: auto;

As we clearly see, the first row is given a height of 100px, while the second row is not given any specific diemnsion, hence the size is set automatically.
