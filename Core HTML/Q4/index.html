Here are a few HTML tags and their functions:

  1. HTML Heading tags: HTML headings are defined with the < h1 > to < h6 >tags.

  2. HTML Paragraphs tag: HTML paragraphs are defined with the < p > tag.

  3. HTML Link tag: HTML links are defined with the < a > tag.

  4. HTML Image tag: HTML images are defined with the < img > tag.

  5. HTML Break tag: HTML < br > tag imposes a break in the paragraph and moves the rest of the paragraph to the next line.
